Command-line utilities for `npm`.

## Install

```sh
npm install ybiq --save-dev
```

## Usage

### CLI

For details, please show help.

```sh
ybiq --help
```

### `init`

Setup Node.js project.

```sh
ybiq init
```

### Programmatic API

```js
const ybiq = require('ybiq')
```
